# Gestión de RUTINAS para el GYM

## Descripción

Este proyecto es una aplicación web que nos permite gestionar rutinas para el gym con ejercicios, series repeticiones, etc...  
También dispone de un registro de usuarios para que cada uno tenga sus rutinas personales.

## Como ejecutar el proyecto

### Instalar dependencias
```bash
pip install django-crispy-forms  
pip install crispy-bootstrap5
```

### Iniciar app
```bash
pip python3 manage.py runserver
```

### Rutas de la aplicación

- **Panel administracion**: http://127.0.0.1:8000/admin/
- **Aplicación**: http://127.0.0.1:8000/
